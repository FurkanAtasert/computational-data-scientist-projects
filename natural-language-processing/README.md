# Natural Language Processing

This folder contains Natural Language Processing (NLP) projects developed in Python, with a focus on text preprocessing, representation, topic modeling, and language-related analysis.

The projects in this section explore different NLP tasks, methods, and workflows, ranging from preprocessing pipelines to comparative modeling experiments and text-based analytical studies.

---

## Scope

This folder includes projects related to:

- text preprocessing
- feature extraction
- topic modeling
- word and document representation
- semantic similarity analysis
- Turkish NLP applications
- applied NLP experiments

## Project Overview

### country_similarity_analysis/
A project focused on similarity analysis between countries using text-based or representation-based approaches.

### lsa-topic-modeling/
A project based on Latent Semantic Analysis (LSA) for discovering topics and latent structures in textual data.

### morphological_ambiguity_classification/
A comparative NLP project for Turkish morphological ambiguity classification using TF-IDF, BERT embeddings, Random Forest, and deep learning methods.

### turkish-text-preprocessing-stanza/
A Turkish text preprocessing project using the Stanza library for linguistic processing and text normalization tasks.
