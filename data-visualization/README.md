# Data Visualization

This directory contains my data visualization projects developed in Python, with a focus on exploratory analysis, analytical storytelling, and multivariate interpretation.

The projects in this section are designed to show how visualization can be used not only to present data, but also to explore distributions, compare groups, detect anomalies, reveal structure, and support statistical modeling workflows.

## Scope

This folder covers three main areas:

### 1. Exploratory Visualization
This section includes projects focused on understanding data through basic and intermediate visual exploration techniques.

Main topics include:

- univariate and bivariate analysis
- categorical and numerical comparisons
- demographic summaries
- missing data inspection
- outlier analysis
- bar-chart based reporting

### 2. Applied Visual Analytics
This section includes projects where visualization is applied to specific datasets in order to extract interpretable and domain-relevant insights.

Main topics include:

- dataset-specific visual analysis
- feature-level comparisons
- pattern discovery
- analytical interpretation through plots

### 3. Multivariate Modeling and Visualization
This section focuses on higher-dimensional analysis where visualization supports statistical modeling and structure discovery.

Main topics include:

- regression-oriented analysis
- principal component analysis (PCA)
- clustering analysis
- multivariate relationship exploration

## Objectives

The main objectives of this directory are:

- to build clear and reproducible visual analysis workflows
- to strengthen statistical reasoning through visualization
- to connect exploratory analysis with modeling processes
- to communicate findings in a structured and professional way

## Tools and Libraries

- Seaborn
- Scikit-learn

## What This Section Represents

This section reflects my approach to data visualization as a data science practitioner.

My general workflow is based on the following principles:

- understanding the data before modeling
- using visualization to validate assumptions
- identifying patterns, trends, and irregular observations
- turning raw analytical outputs into interpretable insights
