# Monte Carlo Simulation

This directory contains my Monte Carlo simulation projects developed for stochastic modeling, probabilistic analysis, and simulation-based decision support.

The projects in this section are designed to demonstrate how simulation techniques can be used to model uncertainty, evaluate probabilistic behavior, test distributional assumptions, and analyze real-world systems under variability.

## Scope

This folder covers two main areas:

### 1. Goodness-of-Fit Tests for Probability Distributions
This section includes projects focused on evaluating how well theoretical probability distributions fit observed data.

Main topics include:

- probability distribution fitting
- goodness-of-fit testing
- statistical compatibility assessment
- distribution-based interpretation

### 2. Bank Service Scenario Report
This section focuses on simulation-based analysis of a service system under uncertainty.

Main topics include:

- queue and service process modeling
- scenario-based simulation
- stochastic system evaluation
- performance interpretation under randomness

## Objectives

The main objectives of this directory are:

- to build clear and reproducible simulation workflows
- to analyze uncertainty through probabilistic modeling
- to evaluate statistical behavior using simulation logic
- to connect theoretical distributions with practical data analysis
- to communicate stochastic results in a structured and professional format

## What This Section Represents

This section reflects my approach to simulation and stochastic analysis as a data science practitioner.

My general workflow is based on the following principles:
