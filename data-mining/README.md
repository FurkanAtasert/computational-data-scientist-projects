# Data Mining

This directory contains my data mining projects developed in Python and related analytical environments, with a focus on pattern discovery, algorithmic experimentation, model evaluation, and text-based data analysis.

The projects in this section are designed to demonstrate how data mining techniques can be used to extract meaningful structure from raw data, evaluate algorithmic behavior, and support data-driven decision making.

## Scope

This folder covers four main areas:

### 1. Algorithm Experiments
This section includes projects focused on testing and comparing algorithmic approaches for solving data-driven problems.

Main topics include:

- algorithm behavior analysis
- experimental comparison
- structured problem solving
- method-oriented evaluation

### 2. Neural Network Backpropagation
This section focuses on the implementation and analytical understanding of neural network learning through backpropagation.

Main topics include:

- neural network fundamentals
- backpropagation mechanism
- learning process analysis
- model training logic

### 3. Performance Evaluation
This section includes projects that assess model or method performance using evaluation-oriented analytical workflows.

Main topics include:

- model performance measurement
- metric-based comparison
- analytical validation
- interpretation of evaluation results

### 4. Text Mining with Zemberek
This section focuses on text mining and natural language processing tasks using Zemberek for Turkish language analysis.

Main topics include:

- Turkish text preprocessing
- tokenization and normalization
- text-based pattern extraction
- language-aware analytical workflows
